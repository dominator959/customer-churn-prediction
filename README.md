# 📉 Customer Churn Prediction

## 📌 Project Overview
A complete machine learning project that predicts customer churn using
classification algorithms. Includes full exploratory data analysis, feature
engineering, model training, and performance evaluation.

## 🎯 Business Problem
A telecom company wants to identify customers likely to cancel their
subscription so they can take action before losing them.

- Which customers are most likely to churn?
- What factors drive churn the most?
- Which ML model predicts churn most accurately?

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| Python 3.x | Core programming language |
| Pandas | Data manipulation |
| Matplotlib / Seaborn | Visualizations |
| Scikit-learn | Machine learning models |
| Jupyter Notebook | Interactive analysis |

## 📁 Project Structure
- data/ — Raw dataset
- notebooks/ — Jupyter analysis and modeling notebook
- outputs/ — Charts and model results
- src/ — Python scripts
- README.md — Project documentation

## 🤖 Models Used
- Logistic Regression
- Random Forest Classifier
- Decision Tree Classifier

## 📊 Key Results
- Best Model: Random Forest
- Accuracy: ~80%+
- Key churn drivers: Contract type, tenure, monthly charges

## ▶️ How to Run
1. Clone this repository
2. Install requirements: pip install -r requirements.txt
3. Open notebooks/churn_analysis.ipynb in Jupyter
4. Run all cells
